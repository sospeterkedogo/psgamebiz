# Design System

## Visual Style
- **High-end dashboard UI**
- Clean, modern, and professional look
- Consistent use of color, typography, and spacing

## Components
- Dashboard cards for key metrics (games played, sessions, revenue)
- Data tables for daily records
- Charts for trends (bar, line, pie)
- Navigation bar (sidebar or topbar)
- Inventory management section
- Tamper-proof log display

## UX Principles
- Clarity: Data is easy to read and interpret
- Tamper-proof: Emphasis on data integrity
- Responsive: Works on desktop and tablet
- Fast access to daily/weekly/monthly views

## Example Color Palette
- Primary: #1A237E (Indigo)
- Accent: #00B8D4 (Cyan)
- Success: #43A047 (Green)
- Warning: #FBC02D (Yellow)
- Error: #E53935 (Red)

## Typography
- Headings: Montserrat, bold
- Body: Roboto, regular

## Iconography
- Use Material Icons or similar for clarity

## Layout
- Grid-based, with clear separation of dashboard, records, and inventory
